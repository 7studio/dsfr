# @gouvfr/grid

La grille proposée par le design system vous permet de structurer vos pages et vos contenus simplement.

## Dépendances
```shell
grid
└─ core
```

## Installation
Installation via NPM:
```
npm install @gouvfr/grid
```
Via Yarn :
```
yarn add install @gouvfr/grid
```
